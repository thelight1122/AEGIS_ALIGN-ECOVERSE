# Interactive 3D Glass Photo Frame (Three.js) — Upload Image + Shape Switch + Drag Rotate + Smooth Zoom - BY gopi chakradhar

A Pen created on CodePen.

Original URL: [https://codepen.io/Gopi-Chakradhar/pen/wBWrxwJ](https://codepen.io/Gopi-Chakradhar/pen/wBWrxwJ).

A clean, minimal Three.js WebGL demo that renders a moving 3D glass object with a photo embedded inside. Includes a modern 2-button UI: Upload Photo + Shape Selector (default Circle, plus Heart, Square, Hexagon, Octagon, Rounded Rectangle). Fully interactive like a real 3D viewer: drag to rotate, scroll/trackpad to zoom, and pinch-to-zoom on mobile. Uses a realistic HDR environment for glossy glass reflections and keeps the photo scale fixed for consistent framing. Perfect reference for 3D product viewers, glass/crystal UI concepts, and interactive image showcases.
