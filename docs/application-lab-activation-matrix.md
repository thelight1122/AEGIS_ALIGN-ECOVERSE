## Activated Behaviors

### `aegis-implementation-apps`

- Header navigation now routes into docs, pricing, security status, and Lab overview surfaces.
- App cards route to the real Application Lab module pages and persist selected-app state.
- Search filters the live app catalog instead of acting as a dead field.
- CTA/footer actions now hand off into docs, profile, pricing, and support paths.

### `aegis-core-shield-features`

- Dashboard and deployment actions now route into active operational/status surfaces.
- Deploy flow stages Core Shield intent and hands off into pricing/unlock.
- Documentation/spec actions route into the Nexus docs portal.
- Search filters reference links and the footer now has working handoffs.

### `aegis-interceptor-features-v2`

- Dashboard/log/rules/support navigation now routes to the corresponding live sections.
- Deploy flow stages Interceptor intent and routes into incident operations.
- Documentation handoff is active.
- Search now filters visible link surfaces.

### `aegis-reflective-prism-features`

- Hero/demo/doc actions now route into live dashboard, docs, and incident workflows.
- Integration/pricing navigation now opens the correct cross-section surfaces.
- Mitigation CTA now routes into the incident center.
- Search now filters visible reference surfaces.

### `aegis-reflective-widget-features`

- Widget starter flow routes into the Developer Depot submission workspace.
- API key CTA now routes into Developer Connect.
- Sales and docs paths are active.
- Copy action now copies a widget snippet when clipboard access is available.
- Search now filters visible reference surfaces.

### `aegis-arbitor-features-1`

- Interceptor/Core Shield/Docs navigation now routes into the correct modules.
- Launch App hands off into Arbitor Consensus.
- Get Started routes into the live Custodian governance surface.

### `aegis-arbitor-features-2`

- Shared Arbitor navigation is active.
- Launch/Get Started route into the live governance voting flow.
- Docs handoff is active.

### `aegis-llm-colab-and-agentic-workshop`

- Colab/workshop/metrics/docs navigation now routes into real live sections.
- Start Collaborating routes into the Agentic Workshop main console.
- Deploy Agents routes into the Workshop creation flow.
