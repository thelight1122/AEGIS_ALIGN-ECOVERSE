Agentic Workshop

Export Workflow Moved

The old bulk export progress modal was a stitched artifact, not a truthful runtime page. Export review is now embedded inside the Detailed Agent View so the Workshop can preserve review notes without inventing transfer percentages, payload sizes, or cryptographic activity.

What Changed

  • Standalone modal route retired from navigation
  • Export review note now lives with the Peer record
  • No synthetic transfer telemetry shown as live state

Current Boundary

A real export runtime can be added later when governed export services exist. Until then, this function remains a bounded record review workflow inside the detailed Peer surface.
