Getting Started with AEGIS Protocol

A comprehensive guide to integrating the AEGIS security layer into your infrastructure. Learn how to secure your nodes, manage identities, and verify cross-chain state transitions.

bolt Quick Installation

The easiest way to get started is by using our CLI tool. Ensure you have Node.js 18+ installed on your system. 

# Install AegisAlign CLI
npm install -g @aegisalign/protocol-cli

# Initialize the security environment
aegis init --provider default

# Generate your local security keys
aegis keys generate --output ./keys/main.key

hub Architecture Overview

The AEGIS Protocol operates as a middleware layer that intercepts and validates all inbound requests before they reach your primary service logic.

Environment Configuration

Add the following environment variables to your .env file to enable protocol communication.

AEGIS_API_KEY Your unique API access token
AEGIS_SECRET_SALT Cryptographic salt for internal hashing
AEGIS_MODE 'strict' | 'permissive' | 'observe'
