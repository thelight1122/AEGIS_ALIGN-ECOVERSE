Simple Implementation

Integrating the AEGIS Reflective Widget takes less than 5 minutes. Use our CDN-hosted script or install via NPM for full type support.

  • check_circle No external CSS required
  • check_circle React and Vue 3 wrappers available

<script src="https://cdn.aegis.com/widget.v2.js"></script>

<div id="aegis-container"></div>

<script>
  AegisReflect.init({
    container: '#aegis-container',
    apiKey: 'YOUR_KEY',
    theme: 'dark',
