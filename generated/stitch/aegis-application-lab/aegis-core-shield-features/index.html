Deployment Quick Start

Activate Core Shield protection across your cluster in under 60 seconds.

bash — aegis-cli
1 curl -sSL https://get.aegis.sh | sh
2 aegis auth login --key=$AEGIS_API_KEY
3 aegis shield enable --policy=hardened
4 # Initializing AEGIS Core Shield...
5 # [SUCCESS] Core Shield active on 12 nodes.
